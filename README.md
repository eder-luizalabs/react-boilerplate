Luizalabs React Boilerplate
==============

# Quickstart

1. `npm install`
1. `npm start`
1. open `http://localhost:3000`

# Testing

1. `npm run test`

# Generating styleguide

1. `npm run styleguide:build`
1. `npm run styleguide:serve`

# Check linting

1. `npm run lint`

# Build to production

1. `npm run build`
